# Student Record Management System (C-Language)

A comprehensive CRUD (Create, Read, Update, Delete) application developed as part of the **BCS101/BCS201: Programming for Problem Solving** curriculum. This project serves as a practical implementation of fundamental and advanced C programming concepts.

## 🚀 Features
- **Dynamic Memory Management**: Uses `realloc` and `free` to manage student records in real-time.
- **File Handling**: Persistent data storage in `.dat` format.
- **Sorting Algorithms**: Implementation of Selection/Insertion sort to rank students by Marks or Roll Number.
- **Advanced Search**: Case-insensitive partial name matching for better user experience.

## 📚 Syllabus Alignment
This project covers the following units:
- **Unit 2**: Arithmetic expressions, conditional branching, and switch statements.
- **Unit 3**: Iterative loops (`do-while`, `for`) for menu navigation.
- **Unit 4**: Structures, character arrays (strings), and sorting/searching algorithms.
- **Unit 5**: Pointers, Dynamic Memory Allocation (DMA), and File I/O.
